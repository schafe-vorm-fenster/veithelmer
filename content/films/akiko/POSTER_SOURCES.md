## Download Instructions

### For poster.jpg:
1. Visit ČSFD.cz gallery link above
2. Find the official cinema poster (portrait orientation)
3. Right-click → "Save Image As" or "Open Image in New Tab" → Save
4. Rename to `poster.jpg`
5. Minimum resolution: 600px width
6. Place in: `content/films/akiko/poster.jpg`

### For teaser.jpg:
1. Look for wide-format promotional still or key art
2. Prefer 16:9 ratio landscape images
3. Should capture key visual or scene from film
4. Rename to `teaser.jpg`
5. Minimum resolution: 1280x720px
6. Place in: `content/films/akiko/teaser.jpg`

## Image Specifications
- Format: JPEG (.jpg)
- Poster: Portrait, minimum 600x900px, 2:3 ratio preferred
- Teaser: Landscape, minimum 1280x720px, 16:9 ratio preferred
- Quality: 80-90% JPEG compression
- Color: RGB color space
- No watermarks or overlay text if possible
