# Project Guidelines

## Content Architecture

### Localization and File Naming
All content bundles must support bilingual localization (German/English).
To ensure consistency across the static site generator, use the following file naming convention for all localized Markdown files:

- **German Content:** `index_de.md`
- **English Content:** `index_en.md`

These files must coexist within the same content directory (e.g., `/content/films/[film-slug]/`).

**Example Structure:**
```text
/content/films/absurdistan/
├── index_de.md  (German metadata and content)
├── index_en.md  (English metadata and content)
└── ...
```

### Frontmatter Schema
All Markdown files must adhere to the following strict Frontmatter schema.

**Required Fields:**

| Field | Type | Description | Allowed Values |
| :--- | :--- | :--- | :--- |
| `title` | String | The official title of the film. | - |
| `language` | String | The language of the file content. | `de`, `en` |
| `director` | String | Name of the director. | - |
| `cast` | List<String> | List of primary cast members. | - |
| `release_year` | Integer | Year of release. | - |
| `duration` | Integer | Runtime in minutes. | - |
| `category` | String | The type of film production. | `Feature`, `Short`, `Commercial`, `Documentary` |
| `poster_image` | String | Relative path to the poster image. | e.g., `poster.jpg` |
| `trailer_video` | String | Relative path to the trailer video (MP4). | e.g., `trailer.mp4` |

**Example:**
```yaml
---
title: "Absurdistan"
language: "en"
director: "Veit Helmer"
cast:
  - "Kristýna Malérová"
  - "Max Mauff"
release_year: 2008
duration: 88
category: "Feature"
poster_image: "poster.jpg"
trailer_video: "trailer.mp4"
---
```
